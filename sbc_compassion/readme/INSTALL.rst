To install this module, you need to install dependencies:
    * requires the following libraries (names from apt-get):
        - openexr
        - libgstreamer0.10-0
        - libgstreamer-plugins-base0.10-0
        - libdc1394-22
        - libavcodec54
        - libavformat54
        - libswscale2
        - libgtk2.0-0
        - python-numpy
        - libv4l-dev
        - zbar-tools
        - python-zbar
        - python-pdfminer
        - python-magic
    * requires opencv3
